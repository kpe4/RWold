## Управление

### Камера
- **WASD** — Перемещение камеры
- **СКМ** — Свободное перетаскивание камеры

### Взаимодействие
- **ЛКМ**:
    - Выбор юнита
    - Растягивание рамки для выбора нескольких юнитов
- **ПКМ** — Приказать выбранным юнитам идти в указанную точку
- **Space** — Снять выделение со всех объектов

### Горячие клавиши
- **Z** — Режим архитектора
- **X** — Режим деструкции
- **C** — Режим рубки
- **T** — Выбрать всех поселенцев на карте
- **H** — Скрыть интерфейс
- **V** — Добыча камня, через круг(зажать лкм)

## Функции для разработчиков

- **P** — Открыть панель управления временем. Позволяет быстро менять текущий час и минуту
- **L** — Включить/выключить туман войны
- **R (Удерживать 2 секунды)** — Полный реген мира
