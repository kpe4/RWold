## Управление

### Мышь
- **ЛКМ (Левая кнопка мыши)**: 
  - Одиночный клик: Выбор персонажа или осмотр плитки земли.
  - Зажатие и перетаскивание: Выделение нескольких персонажей рамкой.
  - Зажатие (в режиме строительства/добычи): "Рисование" задачами по карте.
- **ПКМ (Правая кнопка мыши)**:
  - Команда перемещения для всех выбранных персонажей в указанную точку.
- **Колесико мыши (Middle Click)**:
  - Зажатие и перетаскивание: Панорамирование камеры.
  - Прокрутка: Масштабирование (Zoom) карты.
- **Shift + ЛКМ**:
  - Зажатие и перетаскивание: Альтернативный способ панорамирования камеры.

### Клавиатура
- **W, A, S, D**: Перемещение камеры.
- **Space (Пробел)**: Сброс выделения персонажей.
- **H**: Скрыть/показать интерфейс (с анимацией).
- **T**: Выбрать всех колонистов.
- **Z**: Выбрать режим строительства стен (Architect).
- **X**: Выбрать режим сноса стен (Destruct).
- **C**: Выбрать режим рубки деревьев (Chop).

### Для разработчиков
- **R (удержание 2 сек)**: Регенерация мира.
- **L**: Включить/выключить Туман Войны (Fog of War).
- **P**: Открыть/скрыть панель отладки времени.

## Ветки проекта
- **main**: Основная стабильная ветка.
- **dev_****: Тестовые ветки с бета-функциями.
